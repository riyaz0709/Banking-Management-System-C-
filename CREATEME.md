# Banking Management System

## 📌 Project Overview

The Banking Management System is a console-based application developed in C++ that simulates core banking operations. The project utilizes Object-Oriented Programming principles to manage customer accounts, process transactions, and maintain account information efficiently.

This project demonstrates real-world implementation of OOP concepts such as classes, objects, encapsulation, inheritance, and file handling.

---

## 🚀 Features

- Create New Account
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Fund Transfer
- View Account Details
- Transaction Management
- File-Based Record Storage

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming
- File Handling
- Standard Template Library (STL)

## 💻 Functionalities

### Account Creation
Creates new customer accounts with unique account numbers.

### Deposit
Allows customers to deposit funds into their accounts.

### Withdrawal
Processes withdrawals while checking account balance.

### Fund Transfer
Transfers money between accounts.

### Account Information
Displays customer details and balance information.
